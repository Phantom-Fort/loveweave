# Badge Assets Required

Place 10 PNG files (recommended 64x64 or 128x128, transparent or solid) here:

- first-spark.png      (1 day streak)
- consistent-heart.png (3 day streak)
- weekly-weaver.png    (7 day streak)
- fortnight-flame.png  (14 day streak)
- monthly-bond.png     (30 day streak)
- storyteller.png      (10 stories)
- bridge-builder.png   (50 bridge answers)
- note-keeper.png      (20 notes)
- heart-gardener.png   (love meter >= 50)
- perfect-weave.png    (love meter = 100)

If missing, the UI falls back to emoji icons defined in BADGES in src/main.js.
